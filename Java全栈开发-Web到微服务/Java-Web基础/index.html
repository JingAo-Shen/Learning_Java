🌍 Web应用基础概念

Web应用程序是运行在Web服务器上，通过浏览器访问的应用程序。它基于HTTP协议进行通信，采用B/S（Browser/Server）架构模式。

B/S架构特点：

  • 客户端：浏览器，负责展示用户界面和用户交互
  • 服务器端：Web服务器，处理业务逻辑和数据存储
  • 通信协议：HTTP/HTTPS协议
  • 优势：跨平台、易维护、部署简单

🚀 Java Web技术栈

Java Web开发技术栈：

前端技术：
├── HTML5     (结构)
├── CSS3      (样式)
├── JavaScript (交互)
└── AJAX      (异步通信)

后端技术：
├── Servlet   (核心组件)
├── JSP       (视图技术)  
├── Filter    (过滤器)
├── Listener  (监听器)
└── JSTL      (标签库)

服务器：
├── Tomcat    (轻量级)
├── Jetty     (嵌入式)
└── Undertow  (高性能)

📋 开发环境搭建

  1. JDK安装：Java 8或更高版本
  2. IDE选择：IntelliJ IDEA 或 Eclipse
  3. Web服务器：Apache Tomcat 9.x
  4. 构建工具：Maven 或 Gradle
  5. 数据库：MySQL 8.0
