🌱 Spring框架

企业级应用开发框架，IoC容器和AOP支持

🎯 学习目标

  • 理解Spring核心概念（IoC、DI、AOP）
  • 掌握Spring Bean的生命周期
  • 熟练使用Spring注解
  • 掌握Spring数据访问
  • 理解Spring事务管理

📚 学习路径

1. Spring Core

⏱️ 预计学习时间：2周

2. Spring AOP

⏱️ 预计学习时间：1.5周

3. Spring MVC

Web MVC框架、控制器、视图解析

⏱️ 预计学习时间：2.5周

4. Spring Data

数据访问、JPA集成、事务管理

⏱️ 预计学习时间：2周

5. Spring Security

安全框架、认证授权、权限控制

⏱️ 预计学习时间：2周

6. Spring Boot

快速开发、自动配置、微服务基础

⏱️ 预计学习时间：2周

🔑 核心概念

控制反转 (IoC)

对象创建和依赖关系的控制权由程序转移给容器

依赖注入 (DI)

通过构造器、setter或字段注入依赖对象

面向切面 (AOP)

横切关注点的分离，如日志、事务、安全
