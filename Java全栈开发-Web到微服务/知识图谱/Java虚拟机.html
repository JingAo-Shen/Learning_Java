🔧 Java虚拟机 (JVM)

字节码执行环境，垃圾回收，内存管理

🎯 学习目标

  • 理解JVM的内存结构和运行原理
  • 掌握垃圾回收机制和调优
  • 了解类加载机制和字节码执行
  • 能够进行JVM性能监控和调优

📚 学习路径

1. JVM内存结构

堆内存、栈内存、方法区、程序计数器等

⏱️ 预计学习时间：1周

2. 类加载机制

类加载器、双亲委派模型、类加载过程

⏱️ 预计学习时间：1周

3. 垃圾回收

GC算法、垃圾收集器、内存泄漏排查

⏱️ 预计学习时间：2周

4. 性能调优

JVM参数调优、内存分析、性能监控工具

⏱️ 预计学习时间：2周
