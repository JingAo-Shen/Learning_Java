# 教授个人学术主页

这是一个现代化、响应式的个人学术主页，专为大学计算机科学教授设计。该网站展示了教授的研究领域、个人信息以及所教授的课程。

## 🌟 项目特色

- **个人学术主页**: 现代化设计，展示研究领域和个人信息
- **课程管理系统**: 完整的课程组织结构，支持多学期管理
- **响应式设计**: 完美适配桌面端、平板和移动设备
- **优雅的用户体验**: 平滑动画、悬停效果和直观导航

## 📚 课程体系

### 计算机组成原理 (2024-2025-1学期)
本课程是计算机科学与技术专业的核心基础课程，包含8个主要章节，64学时，4学分：

1. **计算机系统概述**: 计算机发展历程与基本结构
2. **数据表示与运算**: 数制转换、定点数、浮点数运算
3. **存储器层次结构**: 主存、Cache、虚拟存储器
4. **指令系统**: 指令格式、寻址方式、CISC与RISC
5. **中央处理器**: CPU结构、指令执行、流水线技术
6. **总线系统**: 总线结构、仲裁、传输协议
7. **输入输出系统**: I/O接口、中断系统、DMA技术
8. **系统综合**: 性能评价、系统优化

### 行业领域开发Spring Boot (2024-2025-2学期)
本课程包含33个教程模块，涵盖120+学时的学习内容，分为5个技术层次：

1. **基础技能**: Java SE、Git、MySQL、Maven
2. **框架开发**: Spring、Spring Boot、MyBatis、Spring Security
3. **Web技术**: RESTful API、WebSocket、模板引擎、支付集成
4. **系统架构**: 分布式系统、缓存策略、消息队列
5. **工程实践**: 测试、部署、监控、敏捷开发

## 🔬 研究领域

### 主要研究方向
1. **生成式人工智能**: 大型语言模型、扩散模型、多模态生成
2. **EEG大模型技术**: 脑电信号处理、神经网络架构
3. **AI for Science**: 科学计算、天文数据分析、雷达建模
4. **AI赋能科研**: 研究方法创新、科研效率提升

## 🚀 使用指南

### 访问方式
1. **个人主页**: 直接访问 `index.html`
2. **课程学习**: 点击课程卡片进入具体课程
3. **研究项目**: 通过研究领域卡片访问相关项目

### 自定义配置
1. **个人信息**: 修改 `index.html` 中的个人信息部分
2. **课程内容**: 在课程文件夹中添加或修改教程
3. **研究项目**: 更新研究领域的描述和链接
4. **联系方式**: 修改联系信息和社交媒体链接
